# mipscore
使用Verilog实现的32位MIPS架构CPU,并移植ucos到CPU进行运行
# 背景
在进行FPGA的工作当中，为了提高业务水平，锻炼逻辑能力，使用FPGA制作一个MIPS32指令集CPU，并且移植ucos实时操作系统到自制CPU进行运行
